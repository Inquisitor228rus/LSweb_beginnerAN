# Установка
## Установка через gulp
1. склонировать репозитори
2. открыть в консоли папку _(C:/proj/app etc...)_
3. npm i
## Установка DEV
1. npm run gulp
## Установка PROD
1. npm run build
                
## Запуск в браузере
1. npm run gulp

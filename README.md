# установка возможна через gulp скрипт     
## Установка через npm (_деприкетед_)
1. склонировать репозитори
2. открыть в консоли папку _(C:/proj/app etc...)_
3. npm i
4. npm run build
5. npm run scss
## Установка через gulp
1. склонировать репозитори
2. открыть в консоли папку _(C:/proj/app etc...)_
3. npm i
4. npm run gulp
